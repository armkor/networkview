Проект, который с помощью библиотеки WinPcap анализирует весь трафик на одном сетевом интерфейсе, а именно:
- сколько байт (не включая заголовки IP и Ethernet) и пакетов передал каждый из IP адресов;
- список IP-адресов, передавших наибольшее количество байт (5 адресов с наибольшим количеством байт);
- средняя длина пакета;
- для каждого IP адреса отображает список портов TCP и UDP (раздельно), с которого он передавал данные.

Реализовано в OS Windows.
